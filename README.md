# My Video Playground

[구경하기](https://baba-video-playground.web.app)  
[프로젝트 관리 현황](https://trello.com/b/8k4zuHzx/baba-video-playground)

## 기술 스택
> 프레임워크 없이 SPA를 만들어 봅니다.
- html5
- css3
- typescript
- webpack
- babel
- firebase

## 기능
- 영상 컨트롤
  - 시작, 정지, 음소거, 자동재생 등
  - ui부터 기능까지 모두 스스로 구현합니다.
- 이벤트 스택
  - 동영상 요소에서 발생한 event를 쌓아놓는 stack입니다.
